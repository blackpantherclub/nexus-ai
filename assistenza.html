🔑 Come configurare le API Key

Le API key ti permettono di usare i modelli AI direttamente. Sono gratuite e ci vogliono 3 minuti.

ℹ️Le API key vengono salvate solo sul tuo browser. Nessuno le vede, nemmeno noi.

Cos'è un'API Key?

Un'API key è una "password" che ti permette di accedere direttamente ai modelli AI di Anthropic (Claude) e Google (Imagen). Invece di pagare NEXUS AI per ogni richiesta, usi la tua chiave personale — così hai pieno controllo sui costi.

1. API Key di Claude (Anthropic)

1
Vai su console.anthropic.com
Crea un account gratuito con la tua email. Non serve carta di credito.
2
Vai su "API Keys" nel menu
Clicca "Create Key", dai un nome (es. "nexus-ai") e clicca Crea.
3
Copia la key immediatamente
La key viene mostrata una sola volta. Inizia sempre con sk-ant-api03-
COPIA
sk-ant-api03-XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

2. API Key di Google (Imagen 3)

1
Vai su aistudio.google.com
Accedi con il tuo account Google (Gmail). Accetta i termini d'uso.
2
Clicca "Get API Key" → "Create API Key"
Seleziona "Create API key in new project". La key è subito disponibile.
3
Copia la key
Inizia sempre con AIza. Incollala nelle impostazioni di NEXUS AI.
COPIA
AIzaXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
✅Hai la guida interattiva completa nella pagina Configurazione API Key — ti accompagna passo per passo!

Come fare una buona domanda

Più sei specifico, migliore è la risposta. NEXUS AI capisce il contesto, quindi non aver paura di spiegare la situazione completa.

💡Esempio generico: "Scrivimi un sito" → risposta generica
Esempio preciso: "Crea una landing page HTML per una pizzeria a Milano con menu, foto e form contatti" → risposta perfetta

Per il codice

Quando chiedi codice, specifica sempre il linguaggio e il contesto. NEXUS AI sa lavorare con Python, JavaScript, HTML, CSS, PHP, Java, SQL e molti altri.

🖼️ Generare Immagini PIANO PRO+

Come usare Imagen 3 di Google per creare immagini AI professionali.

ℹ️La generazione immagini richiede una Google API Key e il piano Pro o Plus.

Come generare un'immagine

Nella sezione immagini di NEXUS AI, descrivi quello che vuoi con più dettagli possibile. Imagen 3 è molto bravo a seguire descrizioni precise.

Esempi di prompt efficaci

Un logo minimalista per una startup tech, sfondo bianco, colori blu e ciano, stile moderno
Foto realistica di un piatto di pasta italiana, luce soffusa, stile fotografico professionale, alta definizione
Illustrazione digitale di una città futuristica di notte, neon, cyberpunk, dettagliata

Consigli per immagini migliori

  • Specifica lo stile: realistico, illustrazione, minimalista, fumetto
  • Indica i colori principali che vuoi
  • Descrivi la composizione: primo piano, panoramica, dall'alto
  • Aggiungi l'uso previsto: logo, banner, foto prodotto

🔒 Privacy & Sicurezza

Come proteggiamo i tuoi dati e la tua privacy.

Le tue API Key sono al sicuro

Le tue chiavi API vengono salvate esclusivamente nel localStorage del tuo browser. Non transitano mai sui nostri server e nessuno può accedervi — nemmeno noi.

✅API key → salvate solo sul tuo browser · Mai inviate a server esterni

I tuoi messaggi

Le conversazioni vengono inviate direttamente ai server di Anthropic (per Claude) e Google (per Imagen) usando la tua API key personale. NEXUS AI fa da intermediario di interfaccia, non di dati.

I tuoi documenti

I documenti caricati per l'analisi restano solo nella sessione attiva del browser. Chiudendo la finestra, vengono eliminati automaticamente.

Cancellazione dati

Per cancellare tutti i tuoi dati da NEXUS AI, vai nelle impostazioni del browser → Dati del sito → Cancella per nexus-ai.it. Questo rimuove account locale, API key salvate e preferenze.

FAQ

❓ Domande Frequenti

Le risposte alle domande più comuni degli utenti NEXUS AI.

Ho dimenticato di copiare la mia API key. Cosa faccio? ▼
Nessun problema! Vai su console.anthropic.com, entra nella sezione API Keys, elimina quella vecchia e creane una nuova. Ci vogliono 30 secondi. Lo stesso vale per la Google API key su aistudio.google.com.
Ho finito i messaggi del piano Free. Cosa succede? ▼
Il contatore si azzera ogni giorno a mezzanotte. Puoi aspettare il giorno successivo oppure passare al piano Pro per avere 500 messaggi al mese, o al piano Plus per messaggi illimitati.
NEXUS AI funziona su mobile? ▼
Sì! NEXUS AI è ottimizzato per tutti i dispositivi — PC, tablet e smartphone. Apri semplicemente il sito dal browser del tuo telefono (Chrome, Safari, Firefox) e funziona perfettamente.
Posso usare NEXUS AI in inglese? ▼
Assolutamente sì. Anche se l'interfaccia è in italiano, puoi scrivere e ricevere risposte in qualsiasi lingua. Scrivi in inglese e Claude risponderà in inglese.
Le immagini generate sono mie? Posso usarle commercialmente? ▼
Le immagini generate con Imagen 3 sono soggette ai termini d'uso di Google AI Studio. In generale, puoi usarle per uso personale e commerciale, ma ti consigliamo di leggere i termini ufficiali di Google per usi specifici.
NEXUS AI ha un'app da scaricare? ▼
Al momento NEXUS AI è disponibile come applicazione web — si usa direttamente dal browser senza installare nulla. Un'app nativa per iOS e Android è nella nostra roadmap futura.
Come posso richiedere un rimborso? ▼
Se hai sottoscritto un piano a pagamento e non sei soddisfatto, contattaci entro 7 giorni dall'acquisto a supporto@nexus-ai.it. Esaminiamo ogni richiesta e gestiamo i rimborsi entro 3-5 giorni lavorativi.
Perché NEXUS AI usa le API key degli utenti? ▼
Usando la tua API key personale, hai pieno controllo sui costi e sulla privacy. Non dipendi da noi per le chiamate AI — accedi direttamente ad Anthropic e Google. Questo modello, chiamato BYOK (Bring Your Own Key), è sempre più diffuso tra le piattaforme AI moderne.
