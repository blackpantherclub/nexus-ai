Claude API — Anthropic
Per la chat AI, analisi documenti e generazione testi
GRATIS PER INIZIARE
1
Vai su console.anthropic.com
Apri il sito ufficiale di Anthropic per sviluppatori. Clicca su "Sign Up" per creare un account gratuito con la tua email.
🔗 Apri console.anthropic.com
2
Crea il tuo account
Inserisci nome, email e password. Verifica la tua email cliccando il link che ti arriva. Non serve carta di credito per iniziare.
ℹ️ Anthropic regala crediti gratuiti ai nuovi utenti — sufficienti per centinaia di messaggi!
3
Vai su "API Keys"
Una volta dentro la console, nel menu a sinistra clicca su "API Keys". Poi clicca il pulsante "Create Key".
PERCORSO
Console → API Keys → Create Key
4
Copia la tua API Key
Dai un nome alla key (es. "nexus-ai"), clicca Create e poi copia subito la chiave che appare — la vedrai solo una volta!
ESEMPIO
sk-ant-api03-XXXXXXXXXXXXXXXXXXXXXXXX...
⚠️ Copia subito la key e conservala in un posto sicuro. Non la mostrano di nuovo!
5
Incolla qui la tua Claude API Key
Quando hai la key, incollala qui sotto per salvarla in NEXUS AI.
🔑 Inserisci la tua Claude API Key
La key viene salvata solo sul tuo browser, nessuno la vede.
✅ Claude API Key salvata correttamente! Puoi andare al passo successivo.
🎨
Imagen 3 — Google AI Studio
Per generare immagini AI ultra definite con Gemini
PIANO GRATUITO
1
Vai su aistudio.google.com
Apri Google AI Studio — la piattaforma gratuita di Google per sviluppatori. Accedi con il tuo account Google (Gmail).
🔗 Apri aistudio.google.com
2
Accetta i termini
Al primo accesso ti chiede di accettare i termini d'uso. Clicca "Continue" e sei dentro. Non serve nessuna carta di credito.
ℹ️ Il piano gratuito include 60 richieste al minuto per Gemini e accesso a Imagen 3.
3
Clicca su "Get API Key"
In alto a sinistra o nel menu, clicca su "Get API Key" → poi "Create API Key". Seleziona "Create API key in new project".
PERCORSO
AI Studio → Get API Key → Create API Key
4
Copia la tua API Key
La key appare subito. Clicca l'icona di copia per copiarla negli appunti. Inizia sempre con AIza...
ESEMPIO
AIzaXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
5
Incolla qui la tua Google API Key
Salva la key per attivare il generatore di immagini Imagen 3 in NEXUS AI.
🎨 Inserisci la tua Google API Key
Serve per generare immagini con Imagen 3 di Google.
✅ Google API Key salvata! Imagen 3 è attivo in NEXUS AI.
💡 Non hai ancora la key? Puoi saltare questo passo e aggiungerla dopo dalle impostazioni.
